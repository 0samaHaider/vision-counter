**Vision Counter** is a simple AI-powered object detection web app built with **Streamlit** and **YOLOv8**.  
Upload any image, and it will automatically detect all objects and display their counts with annotated bounding boxes.

---

## Features

- Detects multiple objects in an image automatically (no prompts needed).  
- Shows live counts for each detected object.  
- Annotates objects with bounding boxes for easy visualization.  
- Built with **Streamlit** for an interactive web interface.  
- Uses **YOLOv8** for accurate object detection.
